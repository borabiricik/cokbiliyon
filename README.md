# cokbiliyon.com

Tek sayfalık goygoy sitesi. Cümleler `phrases.json` içinden çekilir.

## Başlatma

```bash
npm install
npm run dev
```

Tarayıcıdan `http://localhost:3000` adresine git.

## Katkı

Cümle eklemek için:

1. `phrases.json` dosyasına yeni satır ekle.
2. Cümleleri kısa ve eğlenceli tut.
3. Türkçe karakter kullanımı serbest.

PR atıp gönder, biz de yayına alalım.
